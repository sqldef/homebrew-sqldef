# homebrew-sqldef

Homebrew tap of sqldef

## Usage

```bash
# Install mysqldef
brew install sqldef/sqldef/mysqldef

# Install psqldef
brew install sqldef/sqldef/psqldef

# Install mssqldef
brew install sqldef/sqldef/mssqldef

# Install sqlite3def
brew install sqldef/sqldef/sqlite3def
```
